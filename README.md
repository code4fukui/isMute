# isMute

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An AppleScript for macOS that checks the mute status of Zoom.

## Usage

```sh
# Return 0 if active
osascript zoom-is-active.applescript
```

```sh
# Return 0 if muted
osascript zoom-is-mute.applescript
```
